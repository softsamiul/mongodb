# MongoDB Master

## What is MongoDB?
=> 
    1. Documents based database system
    2. Stores data as JSON-like format

Example: 

{
    name: "Samiul",
    age: 23,
    languages: ["English", "Bangla", "Hindi"]
}

## RDBMS VS MongoDB Terms:
    1. Database = database
    2. Tabels = collections
    3. Rows = documents
    4. Columns = fields
